Welcome to the Leechmission Project

Leechmission is a fast, easy, and free BitTorrent client. In some countries, the dissemination of files like Videos and Music is strictly prohibited. However, downloading this content disseminated by someone else is not illegal. Leechmission is a solution for anyone who wants to obey the local law by disabling sending of content to peers and announcing personal files to the tracker.

WARNING:

  • Leechmission does not send any content to peers, so it stays in strict violation of BitTorrent protocol idea.
  • Leechmission obeys EU regulations about fair use of Videos and Music. It does not send any content to other people, therefore does not violate the law. Remember: downloading is legal, uploading means criminal.
  • Leechmission is slower than other BitTorrent clients - with content uploading disabled you get really low marks in other peers' queues, which blocks your client and don't send you anything.

System requirements

This software is intended to run on Linux machines. It is also possible to build it for BSD and MacOS, but this feature was not tested. There are no plans for supporting Microsoft Windows.
